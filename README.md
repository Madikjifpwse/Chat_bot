# 🎨 Telegram-бот «Центр Красок #1»

AI-ассистент для интернет-магазина centr-krasok.kz.  
Отвечает на вопросы покупателей о компании в формате свободного чата.

## Быстрый старт

### 1. Установите Python 3.10+

Проверьте версию:
```bash
python --version
```

### 2. Установите зависимости

```bash
pip install -r requirements.txt
```

### 3. Создайте Telegram-бота

1. Откройте Telegram, найдите **@BotFather**
2. Отправьте `/newbot`
3. Задайте имя и username бота
4. Скопируйте полученный **токен**

### 4. Получите Anthropic API Key

1. Зайдите на [console.anthropic.com](https://console.anthropic.com)
2. Создайте аккаунт / войдите
3. Перейдите в **API Keys** → **Create Key**
4. Скопируйте ключ

### 5. Настройте переменные окружения

```bash
cp .env.example .env
```

Откройте `.env` и вставьте свои ключи:
```
TELEGRAM_TOKEN=123456789:ABCdef...
ANTHROPIC_API_KEY=sk-ant-...
```

### 6. Запустите бота

```bash
python bot.py
```

Найдите своего бота в Telegram и напишите любой вопрос.

## Как это работает

```
Пользователь → Telegram Bot API → bot.py
                                      ↓
                              История диалога
                                      ↓
                           Anthropic Claude API
                           (системный промпт +
                            база знаний компании)
                                      ↓
                              Ответ пользователю
```

1. Пользователь пишет любое сообщение боту.
2. Бот собирает историю диалога (последние 10 сообщений).
3. Отправляет запрос в Claude с системным промптом, содержащим всю информацию о компании.
4. Claude отвечает **строго на основе базы знаний** — без галлюцинаций.
5. Ответ возвращается пользователю.

---

## Защита от галлюцинаций

Системный промпт содержит жёсткие правила:
- Отвечать только на основе предоставленной базы знаний
- При отсутствии информации — честно сообщать и давать контакты менеджера
- Не выдумывать факты о компании
