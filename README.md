# 🎨 Telegram-бот «Центр Красок #1»

Интеллектуальный AI-ассистент премиум-уровня для интернет-магазина лакокрасочной продукции. Бот функционирует в формате свободного чата, оперативно и экспертно консультируя клиентов по ассортименту, брендам, услугам и контактам компании.

Проект разработан как выпускной пет-проект, полностью настроен и развернут в облачной инфраструктуре.

## 🛠 Технологический стек

* **Язык программирования:** Python 3.11+
* **Фреймворк для бота:** `pyTelegramBotAPI` (Асинхронный режим)
* **Ядро ИИ:** Модели через единый API-шлюз **OpenRouter** (с кастомным системным промптом)
* **Сетевые запросы:** `httpx` (асинхронный HTTP-клиент)
* **Управление окружением:** `python-dotenv`
* **Хостинг / Деплой:** **Railway** (Cloud Infrastructure)
